  ## Description
  This is Ray Takemura's portfolio :)

  ## Installation
  * Clone it by `git clone`
  * Install all the dependencies by typing `npm i` at the root directory (right outside the client folder)
  * Then `cd ./client`
  * Finally, execute `npm start`
